I made two enhancements to my assignment. These are:

1. Flip Card

In the lectures of the unit from week 1 to week 4, the CSS part does not focus on how to animate some elements in the webpages. Therefore, I applied the flippable card to my assignment as a way to improve the user experience. Moreover, this is a good way for some web developers to reveal secret information to the user through placing them behind the card. When the mouse hovers the image, it will be flipped and the information behind the image is revealed

The code I implemented this feature:

  • transform-style:preserve-3d; This CSS line of code is implemented inside the class flip-card-inner to define how the nested elements are rendered in 3D space.
  • transform: rotateY(180deg);This CSS line of code is implemented inside the class of flip-card-inner when the flip-card is "hover". This helps the card to rotate in the vertical direction with an angle of 180 degrees.
  • transform: rotateY(180deg);Again with this code, but now it is placed inside the class .flip-card-back. This helps when the card is no longer hovered and in the position of the back, it will be flipped to the front side.

2. Flexbox

In the lectures of the unit from week 1 to week 4, the CSS part just only mentions the float or relation/absolute position to manipulate the position of each element in the webpage. However, in this assignment, I implemented the flexbox as another to change the position of element in the webpage. The reason why I chose the flexbox instead of float or relative/absolute position is because I want to lay a collection of items in one direction or another. Thanks to the use of flexbox, my product page looks more professional. Each product now has images on the left side and the content on the right side of the page, which helps the reader easily capture the information.

The code I implemented this feature:

  • display:flex; In the product page, I use this code for the class product-item. This class acts as a container of a single TV product. By using this enhancement, it helps the images container and the content container to be flexible in terms of size. This helps them to lay in the horizontal direction of the product container.
  • display: flex;. Moreover, I also implemented this enahancement in the enquire.html page. This helps to lay the input box in one horizontal direction. For example, the first and last name input box are laid in the same line due to the use of flexbox. Furthermore, it is also implemented in the two buttons (submit and reset).
