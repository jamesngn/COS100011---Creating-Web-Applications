I made two enhancements to my assignment 2. These are:

1. AJAX Js for dynamic web page

AJAX stands for Asynschronous JavaScript and XML, which helps the webpage use the XMLHttpRequest object to communicate with servers. It can send and receive information in various formats, including JSOn, XML, HTML and text files. In my assignment 2, I use AJAX Js to connect with JSON myproducts.json. This JSON file stores the information of my TV products in an object format (key-value pair), which helps me easily extend and maintain the webiste in case of having more product items.

The code I implemented this feature:

  • var xhr = new XMLHttpRequest(); This line of code is to create an object XMLHttpRequest, which helps the web page change without reloading the whole page.
  • xhr.open('GET','myproducts.json',true); xhr.send();This line of code is to send a request to a server by using the open() and send() methods of the XMLHttpRequest object.
  • xhr.onload = function()With the XMLHttpRequest object a callback function is created to be executed when the request receives an answer.

2. Timeout for a HTML webpage

The timeout enhancement is about counting the timer behind the webpage. After waiting for a certain period of time, the callback function will be executed.

The code I implemented this feature:

  • const myTimeout = setTimeout(cancelOrder,300000); In the payment page, I set the time to be 5 minutes, which means if the user has not completed the payment form, the function cancelOrder will be called. This function redirects back to the enquire.html. Inside this function has the if statement to clear the timeout. This helps to remove the timer out of the server when a webpage redirection occurs. This function of clearTimeout is similarly used when the user clicks checkout button.
